Description

Omni Responsive Theme is a modern responsive theme with colour switcher featuring 4 colours.
Features
2 Columns
Responsive*, Html5, Css3, 960Grid System
Responsive Navigation Menu (Changes to Selector if Screen is smaller than 768px)
Features Colour Switcher for 4 different colour options. (Blue, Red, Green, Grey)
News Cat Image or CSS News Datestamp Option
Footer Menus with Last Seen Users, About us and Connect

Install

Upload the /themes/ folder to your sites /themes/folder via FTP.
Done!

Usage

SubMenu Usage
In the file Omni/includes/nav.php you can enable or disable the Superfish dropdown menu find $menu = 1; change to 0 to disable it.

To add a dropdown link in header...

  1. In site Links admin Add or edit existing link and add %submenu% before the Link Name and select sub-header only.
  2. All links after this will appear in this dropdown menu.
  3. To end just add %endmenu% in front of the last Link Name in the list.
  4. If you want to make a submenu without the submenu header link to be a link change the Link URL to # it will then make the submenu header link to an anchor

    Note: If %submenu% visibility is set to eg. Members the %endmenu% need to be set to Members as well.

  1. To change Language see /Omni/locale/English.php
  2. To change anything in the footer see Omni/includes/footer.php
  3. To use new Cat image instead of date see the file Omni/includes/render_news-articles.php Find $date = 1; change 1 to 0 to use New Cat Image.
